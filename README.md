# Wanderly - Travel Landing Page

## Overview

Wanderly is a modern and responsive travel landing page built using only HTML and CSS. This project helped me learn how to create a professional website layout, design attractive sections, and improve my frontend development skills.

## Features

* Responsive Navigation Bar
* Fixed Navbar
* Hero Section with Background Image and Text
* Smooth Page Scrolling
* About Section
* Popular Destinations Cards
* Card Hover Effects
* Services Section
* Trip Booking Form
* Contact Section
* Footer Section
* Consistent Color Theme
* Mobile-Friendly Design

## Technologies Used

* HTML5
* CSS3

## What I Learned

Through this project, I learned:

* Creating a complete website structure using HTML
* Styling webpages using CSS
* Adding text and buttons on background images
* Creating a fixed navigation bar
* Implementing smooth scrolling between sections
* Using anchor links for navigation
* Adding hover effects to cards and images
* Designing a clean and attractive color theme
* Creating buttons that navigate to different sections
* Building a professional contact section
* Designing forms using HTML and CSS
* Improving webpage layout and alignment
* Creating a responsive and user-friendly landing page

## Project Purpose

The main goal of this project was to practice HTML and CSS by building a professional travel landing page and understanding how modern websites are designed.

## Conclusion

This project improved my understanding of HTML and CSS and helped me gain practical experience in website design, navigation, hover effects, responsive layouts, and modern UI development.
